### Wrapping

You can set multiple `<V-PlaceloadAvatar />` on a row.
Simply wrap them in `<V-AvatarStack />`

<!--code-->

```vue
<template>
  <V-AvatarStack>
    <V-PlaceloadAvatar class="mx-1" />
    <V-PlaceloadAvatar class="mx-1" />
    <V-PlaceloadAvatar class="mx-1" />
  </V-AvatarStack>
</template>
```
