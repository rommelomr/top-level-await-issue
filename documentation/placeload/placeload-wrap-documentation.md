### Wrapping

You can set multiple `<V-Placeload />` on a row. Simply wrap them in `<V-PlaceloadWrap />`

<!--code-->

```vue
<template>
  <V-PlaceloadWrap>
    <V-Placeload class="mx-2" />
    <V-Placeload class="mx-2" />
  </V-PlaceloadWrap>
</template>
```

<!--/code-->

<!--example-->
