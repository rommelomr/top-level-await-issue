### Size

Control the size of `<V-PlaceloadText />` component to match your interface.
You can set the `height` and the `width` properties according your needs

<!--code-->

```vue
<template>
  <V-PlaceloadText :lines="5" width="75%" last-line-width="25%" />
</template>
```

<!--/code-->

<!--example-->

<V-PlaceloadText :lines="5" width="75%" last-line-width="25%" />
